# 输出格式规范

## 前缀约定

| 角色 | 前缀 |
|------|------|
| 正式回答 | `AI: ` |
| 思考内容 | `[思考过程] ` |
| 工具调用 | `[工具调用] ` |
| 工具结果 | `[工具结果] ` |
| 用户输入 | `你: ` |

## 流式输出流程

1. 先打印 `AI: `
2. 遇到 `reasoning_content` → 换行打印 `[思考过程] `，实时输出思考内容
3. 遇到普通 `content` → 在思考内容之后打印 `---` 分隔线，再打印 `AI: `，实时输出正式内容
4. 流模式下思考内容与正式内容**分阶段**实时输出

非流模式下，思考内容先打印 `[思考过程]`，再打印 `---`，最后打印 `AI: <正式回答>`。

## 特殊命令

| 输入 | 行为 |
|------|------|
| `退出` / `exit` | 结束程序 |
| `对话历史` / `history` | 打印 `MESSAGES` 列表 |
| `debug` | 打印 `RESPONSES` 原始响应 |
| `tokens` | 打印 Token 用量统计（本次会话 + 历史累计） |
